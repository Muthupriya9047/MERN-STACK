CROP PRODUCTION AI&YOLOV8

Agricultural productivity is impacted by soil quality and pest infestations, necessitating effective crop protection.
This project integrate(AI) and deep learning to enhance pest identification and crop recommendation.
The system multiple modules, including soil data analysis, pest detection using YOLOv8, and AI-driven recommendations.
The system also incorporates updating the YOLOv8 model with new pest data to improve detection accuracy.
